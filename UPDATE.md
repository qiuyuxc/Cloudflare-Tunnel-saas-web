# 更新日志

## v1.5.0

### 新增
- 隧道详情页：查看隧道信息与已发布应用程序路由（PC 双列 / 手机单列自适应）
- 应用程序路由管理：新增路由、修改路由（主机名 + 服务地址），catch-all 规则不可编辑
- Custom Hostname 防重复：绑定域名时自动检测已存在的 Custom Hostname，有则更新无则创建
- Ingress 规则去重：重复绑定同一域名不再产生重复路由

### 优化
- 全局设置页移除「转发地址」卡片（该设置保留在域名绑定页）

## v1.4.0

### 新增
- 全局动画系统（基于 uDESIGN.md）：页面过渡、卡片入场、悬停效果、加载状态
- 交错入场动画：控制面板 / 隧道列表 / 设置 / 账户 页面
- 登录页卡片入场动画 + 错误抖动效果
- 状态指示灯脉冲动画（配置未完成时）
- prefers-reduced-motion 无障碍支持
- CLI 密码重置命令：`--reset-password` / `--set-password=<value>`

### 修复
- 页面切换闪烁：用重叠交叉淡入淡出替代 mode="out-in"

### 优化
- 动画时长全局调优（减速 30-50%），过渡更自然
- Docker 构建实时输出：BuildKit + 缓存挂载 + verbose 标志
- Go 构建镜像更新至 1.26
- install.sh 添加密码重置命令说明

## v1.3.0

### 新增
- GitHub Actions 自动构建，镜像推送到 GHCR
- 一键部署：`curl -sO .../install.sh && bash install.sh`
- 支持 `ADMIN_PASSWORD` 环境变量自定义管理员密码
- GHCR 响应超时自动切换 nju.edu.cn 镜像加速

### 优化
- install.sh 安装/更新二合一，更新只需 `./install.sh`
- docker-compose 优先拉取预构建镜像，失败自动回退本地构建
- 无需 git/node/go，只需 Docker 即可部署

## v1.2.0

### 新增

- 域名绑定页支持直接编辑转发地址和优选 CNAME

### 优化
- 控制面板「转发地址」卡片跳转改为域名绑定页面
